This result demonstrates a basic application of path planning and tracking control for the Bitcraze Crazyflie 2 quadrotor model in MuJoCo. The system employs the RRT algorithm combined with B-spline smoothing for path generation, and a Cascade PID controller for precise trajectory tracking.
